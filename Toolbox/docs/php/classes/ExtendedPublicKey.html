__construct()

__construct(string  $version, string  $depth, string  $fingerprint, string  $childNum, string  $chainCode, string  $pubKey, string  $encodedExtPubKey = null, array<mixed,string>  $parentIndeces = array())

Constructor for ExtendedPublicKey objects

Constructor should not be used directly. See \ExtendedPublicKey::fromEncoded() or \ExtendedPublicKey::fromParent() for object creation.

Parameters

string $version

Hexadecimal version (4 bytes), must be a valid version defined in the VERSIONS array
string $depth

Hexadecimal depth (1 byte), must be >= 3
string $fingerprint

Hexadecimal fingerprint (4 bytes), the first 4 bytes of the Hash256 of parent's public key
string $childNum

Hexadecimal child number (4 bytes), the index at which this key is derived
string $chainCode

Hexadecimal chain code (32 bytes), the chain code for this extended public key
string $pubKey

Hexadecimal public key (33 bytes), in compressed DER encoding
string $encodedExtPubKey

(optional, default = null) Hexadecimal encoding of this extended public key, calculated if not given
array<mixed,string> $parentIndeces

(optional, default = []) Array of all parent's child numbers for knowing the derivation path

Throws

\Exception

if the given values are not valid for an extended public key

fromEncoded()

fromEncoded(string  $encodedExtPubKey) : \ExtendedPublicKey

Decodes an Account-level extended public key into an ExtendedPublicKey object representation

Decodes the given Account-level extended public key (in Base58Check encoding) and returns an \ExtendedPublicKey object representing its data. This function should only be used on Account-level keys, as parental data is needed to know the derivation path.

Parameters

string $encodedExtPubKey

Account-level extended public key in Base58Check encoding

Throws

\Exception

if the given extended public key is not valid or not at the Account level

Returns

\ExtendedPublicKey —

Object representation of the given extended public key

fromParent()

fromParent(string  $childNum, string  $chainCode, string  $pubKey, \ExtendedPublicKey  $parent) : \ExtendedPublicKey

Creates a child ExtendedPublicKey object from a given parent object

Creates an \ExtendedPublicKey object that has the fingerprint value calculated based on the given parent, and all child number indeces appended to the returned object.

Parameters

string $childNum

Hexadecimal child number (4 bytes) of the ExtendedPublicKey to be created
string $chainCode

Hexadecimal chain code (32 bytes) of the ExtendedPublicKey to be created
string $pubKey

Hexadecimal public key (33 bytes) of the ExtendedPublicKey to be created, in compressed DER encoding
\ExtendedPublicKey $parent

Parent extended public key for the child ExtendedPublicKey to be created

Object representation of the child extended public key
