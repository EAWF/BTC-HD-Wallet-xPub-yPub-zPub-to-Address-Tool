encode_bech32()

encode_bech32(string  $hexData) : string

Encodes a hexadecimal string into Bech32 Check encoding

Encodes hexadecimal data into the Bech32 encoding format, including a checksum. Does NOT accept non-hexadecimal values. Hexadecimal input is interpreted as bytes, not as a UTF-8 string.
